Bobcat Access

This project aims to help disabled individuals find the locations of ramps near university buildings would serve as a valuable accessibility tool for students, staff, and visitors. This platform would provide a user-friendly interface to easily access important information about building accessibility, particularly for those with mobility impairments.

It will help disabled people to find location of ramps, elevators, and sliding doors.

Elevator

Elevator Locations: The website would highlight the locations of elevators on the interactive campus map, ensuring users know where to find them in each building. Each elevator could be identified with clear markers showing whether it's accessible.

Sliding Doors

Location Identification: The website can include a map and listings of sliding door locations at each building, particularly near entrances and exits, which are key for individuals with mobility challenges. Users could easily see which buildings or areas feature automatic or sliding doors.
